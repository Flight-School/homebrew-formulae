# Flight School Homebrew Formulae

```terminal
$ brew tap flight-school/formulae
```

## Currently On Tap

- **[ner]**:
  A command-line utility for performing named entity recognition on text.
- **[pos]**:
  A command-line utility for tagging part of speech for words in text.
